# Spec - 规范层

## 1. 功能列表
### 布局说明
左右布局，左边显示菜单栏，右边显示内容，弱导航，类似后台管理网站

### 核心功能
| 功能模块 | 功能点 | 优先级 | 状态 |
|----------|--------|--------|------|
| 首页 | 显示网站说明等 | P0 | 待开发 |
| 导航 | 按游戏分，导航层级限制为两级一级：游戏二级：分类，一级菜单仅作为分组，不可点击 | P0 | 待开发 |
| 文章 | 文章目录（TOC），文章内容支持插入视频链接 | P1 | 待开发 |

### 功能扩展预留
全局公告通知

## 2. 用户旅程
访问首页 → 看到网站说明 → 在左侧挑选具体游戏名 → 点击进具体分类 → 阅读文章内容，通过分类/标签发现更多相关内容
**关键触点：**
- 首页加载速度 < 2s
- 文章列表需展示：标题
- 文章详情页需展示：标题、正文、目录

## 3. 验收标准
### 功能验收
#### F-001：首页文章列表
- [ ] 显示网站说明

#### F-002：导航
- [ ] 按游戏名分菜单，游戏下有二级菜单
- [ ] 一级菜单支持展开/收起
- [ ] 一级菜单默认展开第一个游戏
- [ ] 一级菜单仅作为分组，不可点击
- [ ] 点击二级菜单后，右侧展示该分类下的文章列表
- [ ] 文章列表按发布时间倒序
- [ ] 

#### F-003：文章详情页
- [ ] 正确渲染 Markdown 内容
- [ ] 能够将视频链接转为视频播放
- [ ] 显示文章目录（H2/H3 级别）
- [ ] 如果有滚动，返回顶部按钮

#### F-004：文章列表
- [ ] 顶部展示游戏Banner
- [ ] 列表居中
- [ ] 显示标题，发布日期，描述，封面图(如果没有则默认图占位)
