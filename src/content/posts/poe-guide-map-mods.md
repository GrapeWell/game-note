## 前言

开图是流放之路终局的核心玩法，读懂地图词缀是高效刷图的关键。

## 词缀分类

### 前缀（影响怪物）

- **怪物额外伤害**：危险程度高，脆皮 Build 注意
- **怪物攻速/施法速度**：中等危险
- **怪物额外生命**：降低清图效率，不影响生存
- **额外怪物群**：加怪 = 加收益，好词缀

### 后缀（影响环境）

- **元素抗性削弱**：检查自己抗性是否足够
- **无法偷取**：偷取流 Build 致命
- **无法回复**：大部分 Build 都很难受
- **反射**：物理/元素反射是最致命的词缀

## 各 Build 必跳词缀

| Build 类型 | 必跳词缀 |
|-----------|---------|
| 物理攻击 | 物理反射 |
| 元素法术 | 元素反射 |
| 偷取流 | 无法偷取 |
| CI/低血 | 无法回复 |

## 高收益词缀组合

最赚钱的地图应该包含：
1. 额外怪物群
2. 怪物额外生命（安全且不影响收益）
3. 地图物品数量 / 稀有度提升

## 实用技巧

> 开图前养成看词缀的习惯！死亡的经验惩罚远超跳过一张图的损失。

- 用通货重骰危险词缀
- 优先升级地图品质（提升掉落）
- 腐化地图有概率变成稀有未鉴定（全词缀随机），需要谨慎
