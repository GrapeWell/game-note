## 药剂基础

药剂是流放之路中至关重要的装备栏位，合理的药剂搭配可以大幅提升 Build 的强度。

## 药剂类型

### 生命药剂

- 适合生命型 Build
- 推荐使用**永恒生命药剂**
- 词缀选择：即时回复 > 回复速度

### 魔力药剂

- 大部分 Build 需要一瓶
- 可以用**永恒魔力药剂**
- 也可以通过其他方式解决魔力问题

### 功能药剂

这是药剂系统的核心：

| 药剂 | 效果 | 适用场景 |
|------|------|---------|
| 石化药剂 | 增加护甲 | 通用防御 |
| 翡翠药剂 | 增加闪避 | 闪避型 Build |
| 红玉药剂 | 火抗 + 减少点燃 | 火焰场景 |
| 蓝玉药剂 | 冰抗 + 减少冰冻 | 冰冻场景 |
| 黄玉药剂 | 电抗 + 减少感电 | 感电场景 |

## 药剂词缀

### 前缀（推荐）

- **医者的**（击杀回复充能）
- **化学家的**（减少使用充能）

### 后缀（推荐）

- **铁皮**（增加护甲）
- **迅捷**（增加移速）
- **穿刺**（增加暴击率）

## 通用搭配方案

1. 一瓶生命药剂（保命）
2. 一瓶石化药剂（防御）
3. 一瓶钻石药剂（暴击）
4. 一瓶快银药剂（移速）
5. 一瓶功能药剂（根据 Build 选择）

## 使用技巧

> 养成进图按药剂的习惯，药剂效果在战斗中至关重要！

- 钢琴手法：1-2-3-4-5 一键全按
- 有条件可以使用药剂宏
- Boss 战注意药剂充能管理
