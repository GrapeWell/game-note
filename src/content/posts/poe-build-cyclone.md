## Build 概述

旋风斩角斗士是流放之路中最经典的近战 Build 之一，操作简单、清图流畅，非常适合联盟初期开荒。

## 核心机制

- **旋风斩**持续旋转，移动中即可输出
- 角斗士升华提供大量护甲和格挡
- 利用**感电扩散**实现全屏清怪

## 技能搭配

### 主技能

旋风斩 + 贴身打击 + 蛮力 + 残暴 + 近战物理伤害

### 辅助体系

- 跳跃猛击（移动技能）
- 血怒（攻速 + 偷取）
- 惩戒（诅咒）

## 天赋要点

1. 优先点出双持节点
2. 投资攻速和物理伤害
3. 补充生命和护甲节点
4. 后期点出暴击相关

## 装备优先级

| 部位 | 属性优先级 |
|------|-----------|
| 武器 | 物理DPS > 攻速 > 暴击 |
| 胸甲 | 生命 > 抗性 > 护甲 |
| 头盔 | 生命 > 抗性 |
| 手套 | 攻速 > 生命 > 抗性 |

## 预算参考

- 入门版本：约 50C 即可成型
- 进阶版本：约 5D 可刷红图
- 毕业版本：约 30D 全内容通杀
